# 梅花心易 - 靈動六爻系統 (Plum Blossom Divination - Dynamic Hexagram System)
「不動不占，不因事不占」— 結合現代網頁互動與傳統易經數理的動態占卜系統。

線上體驗
https://lunarlunar.github.io/PlumSix/


## 介紹 (Introduction)
梅花心易（又稱梅花易數）是由北宋易學大師邵雍所創的占卜法，強調「心易之妙」。本專案將傳統的梅花易數起卦邏輯，轉化為具備「身心感應」的網頁互動體驗。使用者需要穩定心神，透過滑鼠按壓陣盤的時間與游標的微小位移，來驅動二進位積爻產生的六十四卦，並給出相應的詳盡解籤。

## 核心特色 (Features)
- **動態狀態機管理**：嚴謹的起卦、鎖死、重置生命週期。
- **感應干擾特效 (Interference FX)**
  - **位移漂移 (Drift)**：模擬心神不寧時的隨機位移，考驗使用者的專注力。
  - **縮放脈衝與動態變色 (Pulse & Color)**：隨機變換頻率，增添視覺層次感。
  - **容錯判定機制**：起卦時滑鼠若偏離陣盤中心範圍過大，將會導致心神飄移而失敗，需重新聚氣。
- **二進位卦象運算**：由下而上（初爻至上爻）進行陰陽累積，精確對應先天八卦理數轉換成 64 卦解析。
- **豐富的解籤資料庫**：內建包含「詩詞、典故、總體運勢、事業財運、健康指示」的卦象資料，並以 JSON 結構完美封裝。
- **防呆與極端情況處理**：針對系統錯誤及不可能出現的極端數理情況，皆有預設的錯誤防護與測試卦象提示。

## 操作邏輯 (How to Use)
1. **儀式啟動**：移動滑鼠至畫面中央的陣盤。
2. **凝神聚氣**：按住滑鼠左鍵，系統將在此期間產生視覺干擾。請務必保持滑鼠穩定維持在中心。
3. **積爻成卦**：經歷 2~5 秒隨機感應後，將自動產出一爻（陰或陽）。反覆執行六次。
4. **解卦鎖定**：六爻齊備後，系統自動將畫面向下捲動，展現您的專屬運勢解析。
5. **快速測試機制 (隱藏彩蛋)**：在畫面上連續點擊任意空白處 8 次，即可解除隱藏的「快速生成模式」按鈕，幫助開發者瞬間產出亂數卦象以確認版面。

## 畫面預覽與技術架構
純前端實作，不依賴任何外部框架：
- **HTML5 & Vanilla CSS**：打造富有深邃玄學氣息的 Dark Mode 漸層與發光特效。
- **Vanilla JavaScript**：掌控核心隨機數理生成、DOM 動態渲染與 Event Listeners。
