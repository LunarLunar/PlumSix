# 梅花易數：數位靈動占卜系統 (Plum Blossom I-Ching Oracle)

> 「太極生兩儀，兩儀生四象，四象生八卦。」— 結合北宋邵雍天人合一思想與現代互動科技的數位命理實踐。

---

## 📖 系統概述 (Introduction)
**梅花易數**（又稱梅花心易）是由北宋易學大師邵雍所創，核心精神在於「心易之妙，存乎一心」。本系統將這套具備數千年歷史的玄學體系轉化為具備「身心感應」的網頁互動體驗。

不同於傳統隨機數產生器，本系統利用使用者起卦時的**按壓時長**、**游標微位移**與**亂數干擾因子**，模擬真實占卜中的「氣場」變化，透過嚴謹的二進位位元積爻，最終生成對應《易經》六十四卦的專業解析。

---

## 🔮 核心命理邏輯 (Methodology)
系統內建深度的玄學運算邏輯，力求提供最專業的命理參考：

- **五行生剋運算**：分析卦象中各爻的五行屬性（金、木、水、火、土），根據生剋關係判斷能量強弱。
- **古詩詞解籤**：每一卦皆配有對應的古詩偈語，結合《易經》原始卦辭與爻辭，提供富含哲理的啟示。
- **體用之辨**：模擬梅花易數特有的「體用」關係，判斷主客、內外之勢。
- **靈動積爻機制**：由初爻至上爻逐步堆疊，體現萬物生長的動態過程。

## 🛠 操作流程解說 (Walkthrough)

### 1. 凝神聚氣與動態起卦
在占卜開始前，使用者需穩定心神。移動滑鼠至陣盤中央並長按，系統將啟動「靈動模式」。

![正在占卜過程](Images/info1.png)
*圖一：起卦過程中，系統會產生動態干擾（位移漂移與脈衝縮放），模擬外界磁場對使用者的干擾，測試起卦者的專注度。*

### 2. 六爻解讀與命運析論
當六爻齊備（執行六次按壓）後，系統將自動演算六十四卦並跳轉至結果區。

![占卜結果解析](Images/info2.png)
*圖二：詳細的結果顯示，包含卦象全名、吉凶判定、古詩解籤以及針對事業、財運、健康的深度分析。*

## ✨ 專業技術特色 (Features)
- **玄學核心引擎**：嚴謹的先天八卦與六十四卦理數轉換邏輯，保證卦象生成的學理正確性。
- **心度感應特效**：使用 Vanilla JS 捕捉細微的輸入軌跡，將物理運動轉化為數位規律。
- **沈浸式美學設計**：採用深邃的玄學色調（Dark Mode），結合高滲透光效與平滑動畫，營造肅穆的占卜氛圍。
- **結構化解籤資料庫**：內建完整的 JSON 命理資料庫，支援精準的關鍵字匹配與多維度分析。

## 💻 驅動技術 (Technology Stack)
本專案堅持「純淨開發」原則，不使用任何外部框架，確保效能與穩定性：
- **HTML5 & Vanilla CSS**：打造富有東方科技感的 UI 佈局與發光視覺效果。
- **Vanilla JavaScript**：掌控核心隨機數生成、狀態機管理與 DOM 實時渲染。
